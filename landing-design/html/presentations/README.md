# Презентации ikamdocs

Enterprise-стиль презентации для печати в PDF.

## Файлы

- **ikamdocs-pitch-investors.html** — для инвесторов (проблема, решение, бизнес-модель, roadmap)
- **ikamdocs-pitch-clients.html** — для клиентов (ценность, функции, интеграции, тарифы)

## Экспорт в PDF

1. Откройте HTML-файл в браузере (Chrome, Edge, Safari)
2. `Файл` → `Печать` (или Ctrl/Cmd + P)
3. В качестве принтера выберите «Сохранить как PDF» / «Microsoft Print to PDF»
4. Сохраните файл

Стили `@media print` оптимизируют вывод для печати на A4.
