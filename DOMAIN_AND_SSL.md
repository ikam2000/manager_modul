# Домен ikamdocs.ru и доступность

## Текущее состояние (по вашим скриншотам)

- **Домен:** ikamdocs.ru зарегистрирован и оплачен до 25.02.2027
- **A-запись:** ikamdocs.ru → 195.133.25.37 (настроена)
- **NS:** ns1.timeweb.ru, ns2.timeweb.ru и др.
- **SSL:** Кнопка «Купить сертификат» — сертификат ещё не установлен

## Почему домен может быть недоступен

1. **Нет SSL** — при переходе по HTTPS браузер блокирует соединение. Попробуйте **http://ikamdocs.ru** (без s).
2. **Сервер не настроен** — на 195.133.25.37 должен работать веб-сервер (Nginx + приложение).
3. **Порты закрыты** — порты 80 и 443 должны быть открыты на фаерволе Timeweb.

## Что сделать

### 1. Проверить HTTP

Откройте в браузере: `http://ikamdocs.ru`

Если страница не открывается — веб-сервер на 195.133.25.37 не настроен или порт 80 закрыт.

### 2. Установить SSL (Let's Encrypt)

На сервере (если уже развёрнуты Nginx и приложение):

```bash
ssh root@195.133.25.37
apt install certbot python3-certbot-nginx
certbot --nginx -d ikamdocs.ru -d www.ikamdocs.ru
```

После этого HTTPS будет работать.

### 3. Альтернатива — платный SSL в Timeweb

В панели Timeweb для домена ikamdocs.ru нажать «Купить сертификат», оформить и активировать. Затем в Nginx указать пути к сертификату.

## DNS

DNS настроен корректно. Распространение занимает до нескольких часов, но с NS Timeweb обычно всё обновляется быстро.
