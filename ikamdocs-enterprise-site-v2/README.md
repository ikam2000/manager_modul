# ikamdocs enterprise site v2

Статическая enterprise-версия сайта с сильным product-led дизайном.

## Что внутри
- `index.html` — главная страница с animated architecture, interactive data map и product UI surface
- `platform.html` — детальная продуктовая страница Platform
- `integrations.html` — детальная страница Integrations
- `security.html` — детальная страница Security
- `assets/styles.css` — общий дизайн-сет
- `assets/app.js` — анимации, mobile menu, interactive schema
- `assets/mock-dashboard.svg` — product UI mockup
- `assets/mark.svg` — логомарк

## Как открыть
Просто откройте `index.html` в браузере.

## Как встраивать
Можно загрузить как статический сайт на Netlify, Vercel, Timeweb или Nginx.
